## 📊 النماذج الجديدة

### 1. نموذج الحضور اليومي `Attendance`

**الملف**: `app/models/worker.py`

```python
class Attendance(db.Model):
    id                  # معرف السجل
    worker_id           # معرف العامل
    attendance_date     # تاريخ الحضور
    is_present          # هل العامل حاضر (منطقي)
    status              # حالة الحضور (حاضر، غياب، مرض، إجازة)
    hours_worked        # الساعات المعملة (افتراضياً 8)
    notes               # ملاحظات إضافية
```

**الحالات المدعومة**:
- `حاضر` - العامل حاضر يعمل
- `غياب` - العامل غائب (يتم خصم يوم من الراتب)
- `مرض` - العامل مريض (لا يتم خصم)
- `إجازة` - العامل في إجازة (مدفوعة)

---

### 2. نموذج الحضور الشهري `MonthlyAttendance`

**الملف**: `app/models/worker.py`

```python
class MonthlyAttendance(db.Model):
    id                      # معرف السجل
    worker_id               # معرف العامل
    year                    # السنة
    month                   # الشهر (1-12)
    
    # بيانات الحضور
    total_days              # إجمالي أيام الشهر
    present_days            # أيام الحضور
    absent_days             # أيام الغياب
    sick_days               # أيام المرض
    vacation_days           # أيام الإجازة
    total_hours             # إجمالي الساعات
    
    # حسابات الراتب
    base_salary             # الراتب الأساسي الشهري
    hourly_rate             # سعر الساعة الإضافية
    overtime_hours          # الساعات الإضافية
    overtime_pay            # أجر الساعات الإضافية
    deductions              # الخصومات (غياب)
    bonuses                 # المكافآت
    net_salary              # الراتب الصافي (محسوب تلقائياً)
```

**حساب الراتب الصافي**:
```
الراتب الصافي = الراتب الأساسي + أجر الساعات الإضافية + المكافآت - الخصومات
```

**حساب الخصومات**:
```
الخصومات = عدد أيام الغياب × (الراتب الأساسي / 30)
```

## 🛣️ المسارات الجديدة

### مسارات الحضور اليومي

| المسار | الوصف | الطريقة |
|--------|-------|--------|
| `/workers/attendance` | عرض الحضور اليومي لتاريخ معين | GET |
| `/workers/attendance/<worker_id>/add` | إضافة سجل حضور جديد | GET, POST |
| `/workers/attendance/<attendance_id>/edit` | تعديل سجل الحضور | GET, POST |
| `/workers/attendance/<attendance_id>/delete` | حذف سجل الحضور | GET |

**المعاملات**:
- `date`: اختياري - تاريخ محدد بصيغة `YYYY-MM-DD`

**مثال**:
```
GET /workers/attendance?date=2026-02-14
```

---

### مسارات الحضور الشهري

| المسار | الوصف | الطريقة |
|--------|-------|--------|
| `/workers/monthly-attendance` | عرض الحضور الشهري | GET |
| `/workers/worker/<worker_id>/salary-report` | تقرير الراتب الشامل | GET |

**المعاملات**:
- `year`: السنة (افتراضياً السنة الحالية)
- `month`: الشهر من 1 إلى 12 (افتراضياً الشهر الحالي)

**أمثلة**:
```
GET /workers/monthly-attendance?year=2026&month=2
GET /workers/worker/5/salary-report?year=2026&month=2
```

## 📱 الواجهات الرسومية

### 1. صفحة الحضور اليومي
**المسار**: `/workers/attendance`

**الميزات**:
- اختيار تاريخ محدد للعرض
- عرض جميع الحاضرين والغائبين وحالاتهم
- إحصائيات يومية (حاضر، غائب، مرض، إجازة)
- قائمة بالعمال الذين لم يتم تسجيل حضورهم
- زر لتسجيل الحضور السريع
- تعديل وحذف السجلات

**الألوان المستخدمة**:
- 🟢 أخضر: الحاضرون
- 🔴 أحمر: الغائبون
- 🟡 أصفر: المرضى
- 🔵 أزرق: الإجازات

---

### 2. صفحة إضافة الحضور
**المسار**: `/workers/attendance/<worker_id>/add`

**حقول النموذج**:
- **تاريخ الحضور** *(مطلوب)*: اليوم الحالي بشكل افتراضي
- **الحالة** *(مطلوب)*: اختر من (حاضر، غياب، مرض، إجازة)
- **الساعات المعملة**: الافتراضي 8 ساعات
- **الملاحظات**: اختياري

**التحقق**:
- منع تكرار التسجيل لنفس التاريخ والعامل

---

### 3. صفحة الحضور الشهري
**المسار**: `/workers/monthly-attendance`

**الميزات**:
- اختيار السنة والشهر
- عرض جدول بكل عامل يحتوي على:
  - عدد أيام الحضور
  - عدد أيام الغياب
  - عدد أيام المرض
  - عدد أيام الإجازة
  - الراتب الأساسي
  - الخصومات
  - الراتب الصافي
- الوصول المباشر إلى تقرير راتب كل عامل
- عرض إجمالي الرواتب الشهرية

**الإحصائيات**:
- إجمالي الحاضرين في الشهر
- عدد العمال
- إجمالي الرواتب الصافية

---

### 4. تقرير راتب العامل الشامل
**المسار**: `/workers/worker/<worker_id>/salary-report`

**أقسام التقرير**:

#### أ) بيانات العامل
- الاسم
- النوع (شهري / ساعي)
- الهاتف والبريد الإلكتروني

#### ب) ملخص الحضور الشهري
عرض بصري لـ:
- أيام الحضور (أخضر)
- أيام الغياب (أحمر)
- أيام المرض (أصفر)
- أيام الإجازة (أزرق)

#### ج) سجل الحضور اليومي
جدول بجميع أيام الشهر يتضمن:
- التاريخ
- الحالة (شارة ملونة)
- عدد الساعات
- الملاحظات

#### د) حسابات الراتب المفصّلة

**الجزء الأول - الحسابات الأساسية**:
- الراتب الأساسي
- الساعات الإضافية
- سعر الساعة الإضافية
- إجمالي أجر الساعات الإضافية

**الجزء الثاني - الخصومات والمكافآت**:
- الخصومات (بسبب الغياب)
- المكافآت
- صافي الخصومات والمكافآت

#### هـ) الراتب الصافي
عرض بارز وكبير للراتب الصافي النهائي مع توضيح الصيغة الحسابية

**💡 ميزة الطباعة**:
- زر "طباعة التقرير" لطباعة التقرير مباشرة
- تنسيق تلقائي للطباعة

## 💻 استخدام النظام

### خطوات تسجيل الحضور

1. **الذهاب إلى قسم الحضور**:
   ```
   العمال → الحضور والغياب
   ```

2. **اختيار التاريخ** (اختياري):
   - التاريخ الافتراضي هو اليوم الحالي

3. **تسجيل حضور عامل جديد**:
   - اضغط على اسم العامل في قائمة "العمال الذين لم يتم تسجيل حضورهم"
   - أو اضغط على رابط "تسجيل الحضور" في صفحة تفاصيل العامل

4. **ملء البيانات**:
   - اختر الحالة (حاضر/غياب/مرض/إجازة)
   - حدد عدد الساعات المعملة
   - أضف ملاحظات إذا لزم الأمر

5. **حفظ**:
   - اضغط زر "حفظ"

### عرض الحضور الشهري والرواتب

1. **الذهاب إلى الحضور الشهري**:
   ```
   العمال → الحضور الشهري
   ```

2. **اختيار السنة والشهر**:
   - حدد السنة والشهر المطلوب
   - اضغط "عرض البيانات"

3. **عرض تقرير العامل**:
   - اضغط على زر "تقرير" في السطر الخاص بالعامل
   - ستظهر تفاصيل كاملة للراتب والحضور

4. **طباعة التقرير**:
   - اضغط زر "طباعة التقرير"
   - استخدم خاصية الطباعة في المتصفح

## 📈 أمثلة الحسابات

### مثال 1: عامل شهري مع غيابات

```
البيانات:
- الراتب الأساسي: $500
- أيام الحضور: 22
- أيام الغياب: 6
- أيام المرض: 2 (غير مخصومة)
- دون ساعات إضافية

الحسابات:
- الخصومات = 6 أيام × ($500 / 30) = 6 × $16.67 = $100
- الراتب الصافي = $500 - $100 = $400
```

### مثال 2: عامل ساعي مع ساعات إضافية

```
البيانات:
- سعر الساعة: $10
- متوسط ساعات يومياً: 10 ساعات × 22 يوم = 220 ساعة
- الساعات الإضافية: 220 - (8 × 22) = 36 ساعة
- مكافأة: $50

الحسابات:
- الراتب الأساسي = 8 × 22 × $10 = $1,760
- أجر الإضافي = 36 × $10 = $360
- الراتب الصافي = $1,760 + $360 + $50 = $2,170
```

### مثال 3: عامل مع خصومات ومكافآت

```
البيانات:
- الراتب الأساسي: $600
- أيام الغياب: 2 (خصم) 
- مكافأة الأداء: $100

الحسابات:
- الخصومات = 2 × $20 = $40
- المكافآت = $100
- الراتب الصافي = $600 - $40 + $100 = $660
```
