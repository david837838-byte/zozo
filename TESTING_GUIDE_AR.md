```bash
# ...
```

✅ **علامة النجاح**: عدم حدوث خطأ + ظهور البيانات

---

#### 2. التحقق من قاعدة البيانات

```bash
# دخول shell التطبيق
python

# في Python shell:
from app import create_app, db
from app.models.account import Account
from app.models.user import User

app = create_app()
with app.app_context():
    # التحقق من الحسابات
    accounts = Account.query.all()
    print(f"عدد الحسابات: {len(accounts)}")
    for acc in accounts:
        print(f"  - {acc.name} (Active: {acc.is_active})")
    
    # التحقق من المستخدمين
    users = User.query.filter_by(is_super_admin=True).all()
    print(f"\nعدد Super Admins: {len(users)}")
    for user in users:
        print(f"  - {user.username} ({user.full_name})")
```

✅ **علامة النجاح**: 
- ✓ وجود حساب افتراضي واحد على الأقل
- ✓ وجود مستخدم super_admin مع is_super_admin=True

### 🌐 الاختبار من الويب

#### 4. تسجيل الدخول كـ Super Admin

```
URL: http://localhost:5000/auth/login

بيانات الدخول:
  اسم المستخدم: super_admin
  كلمة المرور: [تُحدد وقت إنشاء الحساب]

✅ توقع النجاح:
  - إعادة التوجيه لـ Dashboard
  - ظهور قائمة جانبية مع "Super Admin" في الأسفل
```

#### 5. الوصول لصفحة إدارة الحسابات

```
من Dashboard، اضغط على:
  القائمة الجانبية → Super Admin → 🏢 إدارة الحسابات

أو الرابط المباشر:
  http://localhost:5000/super-admin/accounts

✅ توقع النجاح:
  - عرض قائمة الحسابات الموجودة
  - ظهور زر "حساب جديد"
  - جدول بأعمدة: (الاسم، المسؤول، عدد المستخدمين، الحالة، الإجراءات)
```

### 🆕 اختبار إنشاء حساب جديد

#### 6. فتح صفحة إنشاء حساب

```
اضغط على "حساب جديد"

✅ توقع النجاح:
  - ظهور Form مع الحقول:
    □ اسم الحساب
    □ اسم المستخدم (مسؤول)
    □ البريد الإلكتروني
    □ الاسم الكامل
    □ كلمة المرور
```

#### 7. ملأ البيانات وإنشاء الحساب

```
المثال 1 - حساب Ahmed:
  اسم الحساب: Ahmed
  اسم المستخدم: ahmed_admin
  البريد: ahmed@farm.com
  الاسم الكامل: Ahmed Mohamed
  كلمة المرور: Ahmed@12345

اضغط "إنشاء الحساب"

✅ توقع النجاح:
  - رسالة نجاء في الأعلى بخضراء
  - إعادة التوجيه لقائمة الحسابات
  - ظهور الحساب الجديد في الجدول
```

#### 8. إنشاء حساب ثاني

```
المثال 2 - حساب Pierre:
  اسم الحساب: Pierre
  اسم المستخدم: pierre_admin
  البريل: pierre@farm.com
  الاسم الكامل: Pierre Dubois
  كلمة المرور: Pierre@12345

✅ توقع النجاح:
  - نفس النجاح كالمثال الأول
  - وجود حسابي Ahmed و Pierre معاً في الجدول
```

### 👥 اختبار إدارة المستخدمين

#### 9. عرض مستخدمي الحساب

```
في جدول الحسابات، اضغط على أيقونة المستخدمين 👥
(بجانب حساب Ahmed مثلاً)

✅ توقع النجاح:
  - ظهور صفحة "مستخدمو الحساب: Ahmed"
  - جدول يحتوي على مستخدم واحد (ahmed_admin)
  - زر "إضافة مستخدم"
```

#### 10. إضافة مستخدم جديد

```
اضغط "إضافة مستخدم"

ملأ البيانات:
  اسم المستخدم: supervisor_1
  البريد: supervisor@ahmed.farm
  الاسم الكامل: أحمد الرقيب
  كلمة المرور: Sup@12345

اضغط "إضافة المستخدم"

✅ توقع النجاح:
  - رسالة نجاح خضراء
  - عودة لقائمة المستخدمين
  - ظهور المستخدم الجديد (supervisor_1)
```

### 🔐 اختبار العزل والأمان

#### 11. اختبار تسجيل الدخول بحسابات مختلفة

**اختبار A: تسجيل دخول ahmed_admin**

```
العنوان: http://localhost:5000/auth/logout
↓
العنوان: http://localhost:5000/auth/login

بيانات الدخول:
  اسم المستخدم: ahmed_admin
  كلمة المرور: Ahmed@12345

✅ توقع النجاح:
  - التسجيل بنجاح
  - Dashboard يظهر "Ahmed" بجانب الاسم
  - القائمة الجانبية (بدون Super Admin)
  - جميع البيانات تتعلق بـ account Ahmed فقط
```

**اختبار B: تسجيل دخول pierre_admin**

```
تكرار نفس الخطوات لـ pierre_admin

✅ توقع النجاح:
  - التسجيل بنجاح
  - Dashboard يظهر "Pierre" بجانب الاسم
  - جميع البيانات تتعلق بـ account Pierre فقط
```

#### 12. التحقق من العزل الفعلي

```
بينما أنت مسجل كـ ahmed_admin:

1. اذهب لأي قسم (مثلاً: العمال)
2. أضف عامل جديد: "عامل أحمد"
3. اسجل الخروج

الآن سجل دخول كـ pierre_admin:

4. اذهب نفس القسم (العمال)
5. تحقق أنك لا ترى "عامل أحمد"

✅ توقع النجاح:
  - pierre_admin لا يرى أي عمال Ahmed
  - كل حساب له عماله الخاصين
```

### ❌ اختبار معالجة الأخطاء

#### 13. محاولة الوصول بدون صلاحيات

```
سجل دخول كـ ahmed_admin (عادي، ليس super admin)

اذهب للعنوان:
  http://localhost:5000/super-admin/accounts

✅ توقع الفشل:
  - رسالة خطأ بحمراء: "ليس لديك صلاحيات كافية"
  - إعادة التوجيه للـ Home
```

#### 14. محاولة إنشاء حساب باسم مكرر

```
سجل دخول كـ super_admin

اذهب: /super-admin/accounts/create

أنشئ حساب باسم "Ahmed" (موجود بالفعل)

✅ توقع الفشل:
  - رسالة خطأ: "اسم الحساب موجود بالفعل"
  - البقاء في نفس الصفحة
```

#### 15. محاولة إنشاء مستخدم باسم مكرر

```
اذهب لإضافة مستخدم جديد

أنشئ مستخدم باسم "ahmed_admin" (موجود)

✅ توقع الفشل:
  - رسالة خطأ: "اسم المستخدم موجود بالفعل"
```
