## 🎯 الميزات الرئيسية

### 1. قسم العمال
- ✓ إضافة وتعديل وحذف العمال
- ✓ تسجيل الورديات (صباحي/بعد ظهر)
- ✓ تسجيل ساعات العمل مع الموقع (جبل/سهل)
- ✓ تتبع ساعات المحرك وحساب المازوت
- ✓ عمال شهريين وبالساعة

### 2. قسم المخزون
- ✓ تسجيل الأدوية والأسمدة
- ✓ تتبع حركات المخزون (دخول/خروج)
- ✓ إدارة التواريخ الانتهائية
- ✓ متابعة الموردين

### 3. قسم الإنتاج
- ✓ إضافة الأصناف والمحاصيل
- ✓ تسجيل الإنتاج اليومي
- ✓ تتبع استهلاك الأدوية والأسمدة على الأصناف
- ✓ إدارة جودة الإنتاج

### 4. قسم المبيعات
- ✓ تسجيل المبيعات اليومية
- ✓ متابعة دفع العملاء
- ✓ حساب الإيرادات
- ✓ معلومات المشترين

### 5. قسم المحاسبة
- ✓ تسجيل الدخل والمصروفات
- ✓ تصنيف الفئات والمصروفات
- ✓ ربط جميع الأقسام بالمحاسبة
- ✓ تقارير مالية شاملة

### 6. قسم التقارير
- ✓ تقارير شاملة لكل صنف
- ✓ تقارير العمال والراتب المستحق
- ✓ التقارير المالية الشاملة
- ✓ التقارير الشهرية

### 7. قسم الإعدادات
- ✓ إدارة حسابات المستخدمين
- ✓ تعيين الصلاحيات المختلفة
- ✓ إدارة أنواع الصناديق والشراحات
- ✓ تسجيل استخدام الشراحات والكراتين

## 📦 المتطلبات

- Python 3.7+
- pip
- Virtual Environment (اختياري لكن موصى به)

## 🚀 التثبيت

### 1. استنساخ المشروع
```bash
git clone <repository_url>
cd farm_management
```

### 2. إنشاء بيئة افتراضية
```bash
# على Linux/Mac
python3 -m venv venv
source venv/bin/activate

# على Windows
python -m venv venv
venv\Scripts\activate
```

### 3. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 4. إعداد قاعدة البيانات
```bash
python run.py shell
>>> from app import db
>>> db.create_all()
>>> exit()
```

## 🎬 البدء السريع

### طريقة 1: الاستخدام اليدوي

```bash
# تشغيل التطبيق
python run.py

# في متصفح آخر
# زر: http://localhost:5000
```

### طريقة 2: استخدام سكريبت الإعداد

#### على Linux/Mac:
```bash
chmod +x setup.sh
./setup.sh
```

#### على Windows:
```bash
setup.bat
```

### إنشاء حساب مسؤول
```bash
python run.py create-admin
```

## 🔑 المستخدمون والصلاحيات

### أنواع المستخدمين

#### 1. المسؤول (Admin)
- صلاحيات كاملة على جميع الأقسام
- إنشاء وحذف المستخدمين
- تعيين الصلاحيات
- إدارة جميع البيانات

#### 2. مدير قسم
- صلاحيات محددة حسب الدور
- يمكن تعيين صلاحيات محددة لكل مستخدم

### الصلاحيات المتاحة

- `can_manage_workers` - إدارة العمال
- `can_manage_inventory` - إدارة المخزون
- `can_manage_production` - إدارة الإنتاج
- `can_manage_sales` - إدارة المبيعات
- `can_manage_accounting` - إدارة المحاسبة
- `can_manage_reports` - الوصول إلى التقارير
- `can_edit` - صلاحية التعديل على جميع البيانات
- `can_delete` - صلاحية الحذف على جميع البيانات

## 🛠️ التكنولوجيا المستخدمة

- **Framework**: Flask 2.3.0
- **Database**: SQLite (مع إمكانية التبديل إلى PostgreSQL)
- **ORM**: SQLAlchemy
- **Authentication**: Flask-Login
- **Frontend**: Bootstrap 5
- **Language**: Python 3.7+

## 📝 الملاحظات والإرشادات

### تغيير قاعدة البيانات
لتغيير من SQLite إلى PostgreSQL:

```python
# في config.py
# تغيير:
# SQLALCHEMY_DATABASE_URI = 'sqlite:///farm_management.db'
# إلى:
# SQLALCHEMY_DATABASE_URI = 'postgresql://user:password@host:port/farm_management'
```

### استخدام Docker

```bash
# بناء الصورة
docker-compose build

# تشغيل الحاويات
docker-compose up -d

# إيقاف الحاويات
docker-compose down
```
