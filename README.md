## نظام إدارة المزرعة المتكامل

نظام شامل لإدارة جميع جوانب عمليات المزرعة باستخدام Flask و Python.

### الميزات الرئيسية:

1. **قسم العمال** - إدارة العمال والورديات وتسجيل ساعات العمل
2. **قسم المخزون** - تسجيل وإدارة الأدوية والأسمدة
3. **قسم الإنتاج** - إدارة الأصناف والمحاصيل والإنتاج
4. **قسم المبيعات** - تسجيل ومتابعة المبيعات
5. **قسم المحاسبة** - إدارة المعاملات المالية والمصروفات
6. **قسم التقارير** - تقارير شاملة لجميع الأقسام
7. **قسم الإعدادات** - إدارة المستخدمين والصلاحيات

### متطلبات التثبيت:

```bash
pip install -r requirements.txt
```

### إعداد قاعدة البيانات:

```bash
python run.py shell
>>> from app import db
>>> db.create_all()
>>> exit()
```

### إنشاء حساب مسؤول:

```bash
python run.py create-admin
```

### تشغيل التطبيق:

```bash
python run.py
```

ثم زر `http://localhost:5000` في المتصفح.

### الهيكل:

```
farm_management/
├── app/
│   ├── models/          # نماذج قاعدة البيانات
│   ├── routes/          # المسارات والـ views
│   ├── templates/       # قوالب HTML
│   ├── static/          # ملفات ثابتة (CSS، JS)
│   └── __init__.py      # تهيئة التطبيق
├── config.py            # إعدادات التطبيق
├── run.py               # ملف البدء
└── requirements.txt     # المكتبات المطلوبة
```

### المستخدمون والصلاحيات:

- **مسؤول (Admin)**: صلاحيات كاملة على جميع الأقسام
- **مستخدمون عاديون**: صلاحيات محددة حسب الدور

### الأقسام والميزات:

#### 1. قسم العمال
- إضافة وتعديل وحذف العمال
- تسجيل الورديات (صباحي/بعد ظهر)
- تسجيل ساعات العمل مع الموقع (جبل/سهل)
- تتبع ساعات المحرك وحساب المازوت

#### 2. قسم المخزون
- تسجيل الأدوية والأسمدة
- تتبع حركات المخزون (دخول/خروج)
- إدارة التواريخ الانتهائية
- تتبع الموردين

#### 3. قسم الإنتاج
- إضافة الأصناف والمحاصيل
- تسجيل الإنتاج اليومي
- تتبع استهلاك الأدوية والأسمدة
- إدارة جودة الإنتاج

#### 4. قسم المبيعات
- تسجيل المبيعات اليومية
- متابعة دفع العملاء
- حساب الإيرادات
- معلومات المشترين

#### 5. قسم المحاسبة
- تسجيل الدخل والمصروفات
- تصنيف الفئات
- ربط جميع الأقسام بالمحاسبة
- تقارير مالية شاملة

#### 6. قسم التقارير
- تقارير لكل صنف
- تقارير العمال
- التقارير المالية الشاملة
- التقارير الشهرية

#### 7. قسم الإعدادات
- إدارة حسابات المستخدمين
- تعيين الصلاحيات
- إدارة أنواع الصناديق والشراحات
- تسجيل استخدام الشراحات

### التكنولوجيا:

- **Backend**: Python Flask
- **Database**: SQLite (قابل للتبديل)
- **Frontend**: Bootstrap 5
- **Authentication**: Flask-Login
