# نظام إيصالات العمل اليومية
# Daily Work Receipts System

## نظرة عامة

تم إضافة نظام شامل لإدارة إيصالات العمل اليومية يسمح بتسجيل ساعات العمل وطباعة الإيصالات بصيغة 50 × 50 ملم.

## الميزات الرئيسية

### 1. **إنشاء إيصالات العمل**
- تسجيل بيانات العامل والموقع والتاريخ والساعات
- دعم أوقات البداية والنهاية
- حساب تلقائي للراتب اليومي
- إنشاء رقم إيصال فريد

### 2. **إدارة الإيصالات**
- عرض قائمة شاملة للإيصالات
- تصفية حسب العامل والتاريخ وحالة الطباعة
- تعديل الإيصالات
- حذف الإيصالات

### 3. **طباعة الإيصالات**
- تصميم احترافي بصيغة **50 × 50 ملم**
- يضمن الطباعة على ورق ملصقات قياسي
- يحتوي على جميع المعلومات المطلوبة:
  - **اسم المؤسسة**
  - **اسم العامل**
  - **التاريخ والوقت**
  - **الموقع ونوع الوردية**
  - **عدد الساعات والراتب**

### 4. **لوحة التحكم**
- عرض إحصائيات يومية وأسبوعية
- ملخص ساعات العمل والمدفوعات
- عرض آخر الإيصالات المسجلة

### 5. **APIs للتكامل**
- API للحصول على بيانات العامل
- إحصائيات يومية وشهرية بصيغة JSON

## قاعدة البيانات

### نموذج DailyWorkReceipt

```python
class DailyWorkReceipt(db.Model):
    id                  # معرف الإيصال
    account_id          # معرف الحساب/المؤسسة
    worker_id           # معرف العامل
    receipt_date        # تاريخ الإيصال
    receipt_number      # رقم الإيصال الفريد
    worker_name         # اسم العامل
    workplace_name      # اسم مكان العمل
    work_date           # تاريخ العمل
    start_time          # وقت البداية
    end_time            # وقت النهاية
    daily_hours         # عدد الساعات
    hourly_rate         # سعر الساعة
    total_payment       # الراتب الإجمالي
    work_location       # الموقع (جبل/سهل)
    shift_type          # نوع الوردية
    notes               # ملاحظات
    is_printed          # هل تم الطباعة
    printed_at          # وقت الطباعة
    created_at          # تاريخ الإنشاء
    updated_at          # تاريخ التحديث
```

## كيفية الاستخدام

### 1. الدخول إلى النظام

اذهب إلى **القائمة الرئيسية** واختر **إيصالات العمل**

### 2. إنشاء إيصال جديد

1. اضغط على زر **"إنشاء إيصال جديد"**
2. اختر العامل من القائمة
3. أدخل البيانات التالية:
   - **اسم مكان العمل** (مثال: المزرعة الرئيسية)
   - **موقع العمل** (جبل أو سهل)
   - **تاريخ العمل** (التاريخ الذي تم العمل فيه)
   - **نوع الوردية** (صباحي/مسائي/كامل)
   - **وقت البداية والنهاية** (اختياري)
   - **عدد الساعات** (مثال: 8 ساعات)
   - **ملاحظات** (اختياري)

4. اضغط **"حفظ الإيصال"**

### 3. عرض الإيصالات

- اذهب إلى **"قائمة الإيصالات"**
- استخدم الفلاتر للبحث حسب:
  - العامل
  - الفترة الزمنية
  - حالة الطباعة

### 4. طباعة الإيصال

1. اختر إيصال من القائمة
2. اضغط على أيقونة **"الطباعة"** (🖨️)
3. اختر **"اطبع الإيصال"**
4. حدد الطابعة والخيارات:
   - **حجم الورقة**: A4 أو مخصص
   - **الهوامش**: صغيرة جداً
   - **الاتجاه**: عمودي

> **ملاحظة**: الإيصال مصمم للطباعة على ملصقات بحجم **50 × 50 ملم**

### 5. تعديل الإيصال

1. اختر الإيصال
2. اضغط على أيقونة **"تعديل"** (✏️)
3. عدّل البيانات المطلوبة
4. اضغط **"حفظ التعديلات"**

### 6. حذف الإيصال

1. اختر الإيصال
2. اضغط على أيقونة **"حذف"** (🗑️)
3. أكّد الحذف

## إحصائيات وتقارير

### لوحة التحكم تعرض:

- **إيصالات اليوم**: عدد الإيصالات المسجلة اليوم
- **إيصالات الأسبوع**: عدد الإيصالات خلال الأسبوع الماضي
- **إجمالي ساعات الأسبوع**: مجموع الساعات المعمولة

## نصائح للاستخدام الأمثل

1. **ملء البيانات بدقة**: تأكد من صحة تاريخ العمل والساعات
2. **استخدام الأوقات**: إذا أمكن، سجل أوقات البداية والنهاية الدقيقة
3. **الملاحظات**: أضف ملاحظات مهمة عن طبيعة العمل
4. **الطباعة القياسية**: استخدم ملصقات 50×50 للحصول على أفضل النتائج
5. **التحديث المنتظم**: حدّث الإيصالات يومياً

## استكشاف الأخطاء

### المشكلة: لا تظهر الإيصالات
**الحل**:
- تأكد من وجود عمال مسجلين
- تحقق من صلاحياتك

### المشكلة: الطباعة غير صحيح
**الحل**:
- استخدم حجم ملصقات 50×50 ملم
- قلل الهوامش في إعدادات الطابعة
- اختبر على ورقة عادية أولاً

### المشكلة: الراتب غير محسوب
**الحل**:
- تأكد من أن العامل له سعر ساعة محدد
- قم بحفظ الإيصال من جديد
